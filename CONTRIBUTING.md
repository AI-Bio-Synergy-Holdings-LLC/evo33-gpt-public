# Contributing

This repository is owned by AI-Bio Synergy Holdings LLC.

## Contribution Control

External contributions are not accepted unless the repository owner has approved the contribution-control method for this repository in the Software/IP Chain-of-Title Register.

Default method: Developer Certificate of Origin (DCO), unless the repository record requires a Contributor License Agreement (CLA), written assignment, or no external contributions.

## DCO Sign-Off

By contributing, you certify that you have the right to submit the contribution under the repository license and that the contribution may be used by AI-Bio Synergy Holdings LLC under that license.

Each commit must include:

```text
Signed-off-by: Your Name <your.email@example.com>
```

## Pull Requests

Before a pull request can be merged:

- The contributor identity must be clear.
- The commit history must include required DCO sign-offs or CLA coverage.
- The contribution must not include third-party code without compatible license evidence.
- Required NOTICE, attribution, dependency, or security records must be updated.
- AI Bio Synergy Inc. usage rights must remain consistent with the intercompany license schedule.
