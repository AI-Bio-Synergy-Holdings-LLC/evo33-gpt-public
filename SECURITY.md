# Security Policy

This repository is maintained by AI-Bio Synergy Holdings LLC.

## Reporting A Vulnerability

Do not report vulnerabilities through public issues.

Send vulnerability reports to:

```text
security@aibiosynergy.com
```

Include:

- Affected repository and version/commit, if known
- Description of the issue
- Steps to reproduce
- Potential impact
- Any suggested mitigation

## Response Process

AI-Bio Synergy Holdings LLC will review the report, triage severity, and coordinate any fix or disclosure process. If AI Bio Synergy Inc. is using the affected software under an intercompany license, Inc. operational and customer-impact review should be coordinated through the intercompany license owner.

## Sensitive Information

Do not include live secrets, private keys, credentials, regulated data, customer data, or regulated health information in public reports, pull requests, commits, or issues.
